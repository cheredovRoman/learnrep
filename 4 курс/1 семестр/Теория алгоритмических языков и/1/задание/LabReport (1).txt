Лабораторная работа 10 (2 часа)
Тема:    Хранение данных в программе интерпретатора
Отчет:
   1) Перечень простых (базовых) типов данных языка программирования.  
   Если по заданию в языке программирования имеются более двух типов   данных, для упрощения дальнейшей реализации рекомендуется выбрать два разных типа.
   2) Для каждого простого типа данных указать в классе семантического дерева описание  поля хранимых значений  данных.  
   3) Если в программе есть не только простые данные, то для всех  специальных данных (структуры, классы, массивы, возвращаемые значения функций и т.д.) указать способ хранения значений.  
   4) Откорректированное  описание класса "семантическое дерево", содержащее  поле хранимых значений  данных.     
   5) Перечень синтаксических диаграмм, в которых выполняется выделение или  освобождение памяти (в отчете должны быть представлены только такие СД).

Рекомендации:

   2) Реализуйте вывод значений данных в операторе присваивания и описания данных.  
   3) Для контроля выделения \ и освобождения памяти.  предусмотреть в режиме DEBUG вывод семантического дерева 
Отчет:
    1) Пример правильного теста с реакцией Вашей программы в двух пежимах:
         - вывод только значений данных при DEBUG == 0,
         - при выполнении операции выделения/освобождения памяти  вывод  семантическое дерево при DEBUG == 1.
       (В программе используются только простые переменные, которым присваиваются  константы или значения других простых переменных. При присваивании рассмотрите пример с приведением типов).
